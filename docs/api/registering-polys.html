PolyMc

The power of a modded server, without a modded client

Registering Polys

You register Polys in your entrypoint using the PolyRegistry class. The PolyRegistry also contains some helper classes, like the CustomModelDataManager that prevents 2 polys using the same value. Look at the PolyRegistry class to see the other classes available.

Item Polys

PolyMc provides quite good support for items out of the box. There may be some cases in which you want to modify the poly used however. In most cases it's best to use the CustomModelDataPoly as a base.

*note: if your modded item has damage, replace CustomModelDataPoly with DamageableItemPoly* Example: changing the item used on the client 

public void registerPolys(PolyRegistry registry) {
    registry.registerItemPoly(moddedItem, new CustomModelDataPoly(registery.getCMDManager(), moddedItem, Items.DIAMOND_SWORD))
}
This will Poly your item as usual, just using a diamond sword instead of a stick. It will automatically allocate a CMD value for it.
